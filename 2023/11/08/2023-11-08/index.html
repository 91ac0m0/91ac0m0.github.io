古老的 networkmanager 修复方法

for ubuntu 23.10

service NetworkManager stop  
// 停止网络服务  
sudo rm /var/lib/NetworkManager/NetworkManager.state  
// 删除相关配置文件  
service NetworkManager start  
// 重启网络

因为异常时关机网络会关闭，这时候需要重启网络。删除 NetworkManager. State 文件正是为了把网络 turn on as default
